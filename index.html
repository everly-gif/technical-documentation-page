Datatypes

The data-type in a programming language is the collection of data with values having fixed meaning as well as characteristics. Example: integer, floating point, character, etc.

Types

  • Primary(Built-in) Data Types
  • Derived Data Types:
  • User Defined Data Types

Primary(Built-in) Data Types include void, integer, character, long decimal(double) and float(decimal).
Derived Data Types include Array, References, and Pointers
User Defined Data Types include Structure, Union, and Enumeration

Here's how to declare built-in datatype using c

int;
float;
char;
char[200]//string
Variables

The primary purpose of variables is to store data in memory for later use. When you declare a variable in C, that means you are asking the operating system to reserve a piece of memory with that variable name.
Some Rules are as follows:

  • A variable name can consist of Capital letters A-Z, lowercase letters a-z, digits 0-9, and the underscore character.
  • The first character must be a letter or underscore
  • Blank spaces cannot be used in variable names.
  • Special characters like #, $ are not allowed.

syntax

type variable_name;
If-else statements

If-else statements in C is also used to control the program flow based on some condition, only the difference is: it's used to execute some statement code block if the expression is evaluated to true, otherwise executes else statement code block.

Important

  • If -else statements can be nested
  • Only if has a condition and not the else part.
// Program to display a number if it is negative #include
int main() {
int number;
printf("Enter an integer: ");
scanf("%d", &number);
// true if number is less than 0
if (number < 0) {
printf("You entered %d.\n", number);
}
printf("The if statement is easy.");
return 0;
}

When the user enters 5, the test expression number is less than zero so it is evaluated to false and the statement inside the body of if is not executed

Arrays

An array is a variable that can store multiple values. For example, if you want to store 100 integers, you can create an array for it.

Key notes

  • Arrays have 0 as the first index, not 1.
  • If the size of an array is n, to access the last element, the n-1 index is used.

Declaration of an array

dataType arrayName[arraySize];
Structures

In C programming, a struct (or structure) is a collection of variables (can be of different types) under a single name.

Access members of a structure

There are two types of operators used for accessing members of a structure.

Example
person.salary

  • . - Member operator
  • -> - Structure pointer operator

Syntax

struct structureName
{
dataType member1;
dataType member2;
...
};
